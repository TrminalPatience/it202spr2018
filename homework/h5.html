  1. How do we include Bootstrap in our pages/sites/apps?
    Answer: We include Bootstrap by making the appropriate calls to the Bootstrap CSS and JS libraries as you can see in the head of this page.

  2. What's a CDN?
    Answer: A content delivery network or content distribution network (CDN) is a geographically distributed network of proxy servers and their data centers.

  3. What's one advantage of using a CDN instead of hosting your own copy of the files?
    Answer: Increases the speed of access and the relative performance by distributing the sourcing of files to several different locations which end users will connect to based on their geographic proximity to the closest server hosting the content.

  4. In RWD, what is a breakpoint?
    Answer: Breakpoints are the point a which your sites content will respond to provide the user with the best possible layout to consume the information.

  5. What breakpoint values does Bootstrap use?
    Answer: Small 576 px and up, Medium 768 px and up, Large 992 px and up, and Extra Large 1200 px and up.

  6. In Bootstrap, what is the difference between fixed width container and a full width container?
    Answer: A fixed-width container changes its max-width at each breakpoint. While a fluid-width container stays 100% wide all the time.

  7. Bootstrap (and many RWD implementations) break the page up into how many columns?
    Answer:The default Bootstrap grid system utilizes 12 columns

  8. In Bootstrap, what's the purpose of a row?
    Answer:To be a space for columns inside containers. It allows for tidier designs.
